## 零、大千世界 —— 三界总纲

三界统称为**大千世界**。人界为基、灵界为阶、仙界为顶，三者以飞升通道相连，被天道法则统御，灵气永恒轮回驱动一切兴衰。

- **人界**：灵气枯竭的末法残烬，弱肉强食的丛林法则。宗门以灵脉为核心争夺资源，散修在夹缝中求生
- **灵界**：灵气浓度三倍于人界，五种族混战的阴谋棋盘。仙界世家操纵种族矛盾，棋子与棋手的界线模糊
- **仙界**：天道压迫下的永恒牢笼。金仙劫无解，仙帝以恐惧统治，四大家族在权力游戏中互相吞噬

大千世界的核心法则：修炼是对天地的掠夺，飞升是窃取，天劫是天道收债。没有任何存在能真正超脱——仙帝不能，大罗金仙不能，天道本身也不能。

## 一、宇宙论 —— 灵气掠夺与天道循环

### 1.1 灵气的本质

灵气是三界的根基性资源，总量恒定，可被生灵吸纳窃取。它不是无穷之物，而是有限的、可被吞噬的。

### 1.2 修炼的真相

修炼的本质是将天地间的灵气强行纳入己身。每一次吐纳、每一次突破，都是在从天地窃取灵气。修为越高，对天地的消耗越大。

- 修炼者是灵气的黑洞——活得越久、修为越高，负担越重
- 修士密集之地灵气必然枯竭——这是因果，不是偶然
- 凡人视修士为掠夺者——他们吸干了大地的灵气，留下一片贫瘠

### 1.3 飞升即是掠夺的极致

飞升者体内囤积的灵气随之离开此界，永远不会回来。人界灵气越来越少，不是因为天道无情，而是因为历代飞升者带走了它。末法时代的成因是被偷走，而非自然衰退。

### 1.4 天劫的真相：天道在收债

天劫不是考验或恩赐，是天道试图收回被窃取的灵气。

- **雷劫**：以雷霆之力轰击肉身，试图将灵气震散归还天地
- **心魔劫**：动摇道心，修为崩溃则灵气自然散逸归天
- 渡劫成功 = 抗住了天道的收债；渡劫失败 = 天道成功收回部分灵气

修士口中的"天道考验"是自我美化，真相是每一次天劫都是抢劫与反抢劫的战争。

### 1.5 金仙劫与大能陨落

大罗金仙每隔万年必须面对金仙劫，且一次比一次凶猛。这是天道对最大债务人的最终清算。**没有金仙能度过九重金仙劫**——这是铁律（太昊打破铁律属极少数例外）。

大能陨落时，毕生灵气一瞬间散尽反哺三界，如同灵气的超级海啸。三界灵气浓度骤升，万物复苏，新的修仙盛世来临——直到被新一代修士再度吸干。

### 1.6 永恒的轮回

```
繁荣 → 修士崛起 → 灵气被吸纳 → 飞升带走灵气 → 枯竭（末法）
  ↑                                                    ↓
  ← 大能陨落灵气反哺 ← 金仙劫灭杀大能 ← 天道追债升级 ←
```

这不是天道恶意，而是三界的免疫系统。修仙者是天地的"癌细胞"，天劫是"免疫反应"，大能陨落是最极端的免疫手段。

### 1.7 灵气大循环的五大劫难

| 纪元 | 绝对年份 | 跨度 | 灵气水平 | 标志性事件 |
|------|---------|------|---------|-----------|
| 鸿蒙劫 | 1 - 10,000 | ~1万年 | 极盛 | 天地初开，首批生灵觉醒灵智 |
| 天衍劫 | 10,001 - 38,000 | ~3万年 | 充盈→衰减 | 修仙始现，首次飞升，天劫首降 |
| 灵衰劫 | 38,001 - 72,000 | ~3万年 | 衰减→稀薄 | 宗门建立，大能陨落首次反哺，正邪大战 |
| 末法劫 | 72,001 - 108,000 | ~4万年 | 稀薄→枯竭 | 仙界世家操纵下界，太昊与青帝证道大罗 |
| 寂灭劫 | 108,001 - | ~9万年 | 枯竭→微苏 | 末法至暗，金仙陆续陨落，灵气微弱复苏 |

### 1.8 当前时代：破晓前的至暗时刻

当前处于**寂灭劫末期**（绝对年份约198,647年），正值灵气大循环的转折期。数位大罗金仙在最近数万年内陆续陨落，灵气浓度从"枯竭"开始缓慢回升，但近十万年的掠夺积累太深——人界灵气从"绝境"爬到了"苟延残喘"。

关键张力：
- 仙界世家感受到金仙陨落的恐惧，加速对下界的掠夺
- 灵界因为仙界幕后操纵者的焦虑而战况更惨烈
- 人界修士隐约感知灵气在变浓，野心萌动——**希望比绝望更能激发杀意**

## 二、人界 —— 末法残烬中的宗门丛林

### 2.1 人界总论

灵气浓度已跌至0.4以下，绝大多数地区无法自然修炼，只有灵脉所在处残存可用灵气。末法时代是数万年缓慢窒息的结果。

**社会法则：弱肉强食是唯一的法。** 资源稀缺，每个修士都是竞争者。宗门战争不为荣耀，为灵脉控制权。散修的储物袋、丹药、灵根都是别人眼中的资源。

### 2.2 五大域

**天元域（中州）**：双灵脉交汇，灵潮复苏震中。天道剑宗与碧云宗争夺灵脉，扩招规模末法以来之最。灵潮峰值 0.80

**沧澜域（东荒）**：废灵脉冒泡的新战场。散修抢先占据，剑宗派兵"收复祖产"。残剑共鸣，剑意觉醒传闻不断。灵潮峰值 0.55

**瘴冥域（南疆）**：灵潮最猛烈处。封印松动，远古妖兽低吼。万毒谷虫蛊最先感知复苏。正道"清剿"实为抢资源。灵潮峰值 0.90

**荒墟域（西极）**：远古遗址随灵潮显现，功法灵器出土。金刚寺独占最大遗址入口。体修不受灵气枯竭影响。灵潮峰值 0.50

**幽寒域（北冥）**：魔修最先感知复苏。血河教宣称"血月将临"大肆扩张。冻土封印松动——不是灵气复苏，是沉睡之物在翻身。灵潮峰值 0.65

### 2.3 宗门生态

末法时代的宗门是以灵脉为核心的武装集团。

- **大宗门**（天道剑宗、碧云宗）：控制最佳灵脉，收徒极苛，末法时代只收天灵根和上品灵根
- **小宗门**：困守二流灵脉，挣扎求生，弟子资质参差但手段更狠辣
- **散修**：最底层，无灵脉无靠山，大多终其一生在练气期徘徊。"修仙路上最危险的，永远是另一个散修"
- **凡人**：被忽视的群体，灵气枯竭的最大受害者。对修士态度复杂：渴望后代踏入仙途，又怨恨修士偷走天地灵气。发现灵根的孩童是宗门征收的"税"

### 2.4 人界核心矛盾

1. **灵脉争夺**：有限资源下的零和博弈
2. **正邪之分的虚伪**：正道垄断灵脉后将不服从者打为"魔道"。北原魔修未必比中州正道更残忍——只是没有灵脉美化自己
3. **飞升的诱惑与恐惧**：飞升带走灵气会加速人界枯竭，但不飞升只在末法中衰亡
4. **灵气复苏的暗涌**：化神期前辈感知灵气微浓，此消息一旦传开将引发比资源争夺更疯狂的东西——希望

## 三、灵界 —— 森罗万象中的棋盘阴谋

### 3.1 灵界总论

灵气浓度是人界三倍，飞升者踏入如同从沙漠到绿洲。但灵界不是天堂——人族修士在这里是最底层的外来者。

灵界的种族战争从来不是种族之间的战争。每一个种族背后都站着一个仙界家族，每一场战争都是仙界权力博弈的代理战争。

### 3.2 灵界种族

**人族（飞升者的艰难立足）**：非灵界原住民，无数纪元飞升修士后代。功法体系最完善，个体战力不俗，但根基最浅、盟友最少。

**妖族（万兽之灵）**：最强大的原生种族。灵气孕育的灵兽觉醒灵智后形成的种族，以血脉尊卑划分阶层。龙族凤族为皇族。内部分裂严重，各族仇恨不亚于对外族敌意。

**灵族（天地之子）**：最神秘种族，从灵气浓度极高处自然凝聚诞生。天生与灵气亲和，能感知天道意志，数量稀少。对灵气枯竭最敏感。

**鬼族（亡者的国度）**：强大修士死后残魂不灭凝聚而成。"国土"在灵界阴影层冥渊。与人族矛盾最深：人族夺舍之术在鬼族看来是最大亵渎，鬼族吞噬生魂在人族看来是最大邪恶。

**魔族（深渊的回响）**：最被孤立也最令人恐惧。起源众说纷纭——域外天魔入侵后裔或上古大能陨落怨念所化。以负面情绪为食，以血祭为仪式。

### 3.3 三域格局

**落尘域（飞升者围城）**：人界飞升者踏入灵界的第一站。灵界之门所在，人族主要势力范围。灵界执法使名义维持秩序，实际是仙界世家的眼线，记录每个飞升者的灵根、资质、修炼方向，上报各自背后的仙界主子。

**万劫域（蛮荒战场）**：真正无序之地，妖族部落、魔族巢穴、鬼族边缘领地交错。没有规则只有暴力。也是机缘最多的地方——无数上古遗迹和远古传承。

**弈天域（棋盘中心）**：灵界核心，大乘修士聚集。表面最文明有序，实际最危险的阴谋漩涡。通天塔矗立于此，飞升仙界唯一通道。但谁能上塔不靠实力，靠你在仙界世家的"分量"。

### 3.4 仙界世家的棋局

| 仙界世家 | 灵界代理人 | 操纵手段 |
|---------|-----------|---------|
| 天枢（仙帝一系） | 灵界执法使体系 | 以"秩序"之名控制飞升通道，筛选有价值的棋子 |
| 紫微（姬氏） | 灵族中的预言者一脉 | 通过"天命预言"引导灵族行动，制造种族对立 |
| 丹霞（姜氏） | 人族最大的丹药商会 | 用丹药供应链控制人族势力兴衰 |
| 玄天（嬴氏） | 妖族中的鹰派势力 | 挑动妖族内部冲突，让妖族永远无法统一 |
| 苍冥（自由领地） | 无固定代理人 | 不参与棋局，但暗中收留被抛弃的棋子 |

核心目的：确保灵界产出的灵材、天才和资源优先输送到自家门下。种族战争是维持灵界分裂、防止任何一方做大的手段。

### 3.5 灵界核心矛盾

1. **种族仇恨的真实与虚构**：历史积淀与仙界世家刻意煽动并存。揭穿真相可能引发更大混乱
2. **飞升通道的垄断**：真正有威胁的修士被暗杀在突破之前，顺从的棋子被送上仙界成为新傀儡
3. **灵族的末日预言**：预言被紫微家扭曲传播，制造恐慌而非警醒
4. **棋子的觉醒**：信赖的导师、盟友甚至道侣，都可能是仙界世家安插的棋子

## 四、仙界 —— 天道牢笼中的永恒棋局

### 4.1 仙界总论

仙界不是终点，是最大的牢笼。仙气浓度越高，天道感知越敏锐，天劫威力越恐怖。金仙劫没有尽头，每隔万年一次且越来越猛，九重之后必死无疑。大罗金仙不过是天道给你的一张限期死刑通知书。你越强，天道的收债越狠。

### 4.2 仙帝与天枢仙域

仙帝·太昊是活了不知多少纪元的大罗金仙（天衍劫中期出生，现约178,000岁），靠着远超常人的手段一次又一次渡过金仙劫。每渡过一重，凌霄宝殿的根基就裂开一道缝。

**太昊的关键时间线：**

| 绝对年份 | 距今 | 事件 |
|---------|------|------|
| ~20,000 | ~178,000年 | 出生（天衍劫中期） |
| ~55,000 | ~143,000年 | 以太乙金仙之身建立凌霄宝殿 |
| ~75,000 | ~123,000年 | 四大家族体系确立 |
| ~105,000 | ~93,000年 | 证道大罗金仙（末法劫后期） |
| ~155,000 | ~43,000年 | 第五次金仙劫（沈牧云动摇的触发点） |
| ~165,000 | ~33,000年 | 第六次金仙劫来临时 → 沈牧云留辞书前往苍冥 |
| ~195,000 | ~3,600年 | 第九次金仙劫（打破铁律） |
| ~205,000 | ~6,400年（未来） | 第十次金仙劫预计 |

**仙帝的统治逻辑不是仁慈，而是恐惧。** 他需要整个仙界为他服务：丹霞的丹药续命，紫微的情报预判天劫时机，玄天的军力震慑异动，苍冥的散仙充当炮灰。对四大家族的态度：让他们互相牵制，谁也不能做大，谁也不能消失。

第九次金仙劫的代价：紫微提前十年预判时机，丹霞倾尽库存炼三百颗续命丹，玄天调十万仙兵布九重防御阵，苍冥征三千散仙充雷劫缓冲——三千散仙无一生还。丹霞库存清空、玄天军力折损三成、苍冥散仙恨之入骨。**四大家族的忠诚出现了第一道裂痕。**

**太昊与五色帝：** 太昊证道大罗时（~105,000年），五色帝中最早的白帝已是大罗（~82,000年证道）。太昊对五色帝的态度从忌惮到轻蔑——白帝和赤帝先后陨落、黄帝和黑帝失踪后，太昊认为"散修终究是散修"。但他不知道黄、黑二人仍活着，更不知道青帝散功后尚在人界。

### 4.3 四大家族

**紫微 · 姬氏（情报与权谋）**：天机阁情报网络覆盖三界，每个化神期以上修士都有一份档案。真正底牌是预言——璇玑藏书阁收藏上古以来的星象记录，观星者能推演天道意志。知道太多的人往往死得最快。

**丹霞 · 姜氏（丹药与财权）**：灵晶矿脉出产七成灵晶，太古丹炉、万宝仙市。真正实力是用财富控制的依赖关系——所有人都是姜氏的"客户"。表面最温文尔雅，实则最冷酷无情。

**玄天 · 嬴氏（军权与边防）**：唯一掌握成建制仙军的家族。家训"以战养战"，当嬴氏拔剑其他家族都会闭嘴。核心困境是域外天魔——仙帝故意让天魔压力保持"恰到好处"，既牵制嬴氏又不让仙界覆灭。

**苍冥（无主的自由）**：散仙、隐仙、妖仙聚居地。表面最自由实际最危险——没有家族庇护意味着任何人都可以在这里消失而不被追查。逍遥城赏金榜、醉仙楼酒后真言、幽渊城妖仙市集。仙帝眼中钉。

### 4.4 域外天魔

混沌虚空中诞生的存在，以吞噬仙气和仙人本源为食。

- **不惧天劫**：不属于三界，天道"收债"对它们无效
- **形态多变**：可化为任何生灵模样渗透仙界。传闻仙界高层中可能已有天魔替身
- **以仙气为食**：仙界对天魔而言是丰盛的猎场

禁忌推测：天魔可能不是外来的敌人，而是天道用来加速金仙陨落的工具——当金仙劫不足以杀死最强者时，天道会"恰好"放任天魔入侵来补刀。

### 4.5 仙界权力游戏（四维棋盘）

**第一维：朝堂政治**——百仙朝堂上的唇枪舌剑，争夺资源分配、律法解释权、新飞升者"领养权"。

**第二维：家族博弈**——暗探渗透、供应链操控、军力控制、黑市暗杀。当两个家族公开翻脸，意味着暗中较量已有结果。

**第三维：仙帝平衡术**——利用四大家族互相牵制维护绝对权威。金仙劫的阴影让仙帝越来越焦虑偏执不可预测。

**第四维：天道的棋局**——真正的棋手是天道。仙帝隐约感觉到天道在用他——用他管理仙界，用四大家族维持分裂，用天魔制造危机——一切都是天道"免疫系统"的一部分。

### 4.6 仙界核心矛盾

1. **永生的幻灭**：飞升仙界后发现永生是注定的败局
2. **权力的本质**：权力斗争是生存竞争——更多资源 = 更大概率渡过下次金仙劫
3. **天道的真相**：一生修炼和牺牲最终都在为天道循环服务
4. **天魔与人心的界限**：当信任本身成为致命弱点，"谁是敌人"比"怎么打赢"更难
5. **苍冥的选择**：放弃修炼停止掠夺的道路意味着被时间杀死。自由还是生存？

## 五、三位主角

### 5.1 游戏结构

三位主角是完全独立的人物。玩家在开局时选择一位，另外两位以NPC身份存在于世界中。是否相遇取决于LLM叙事演化。所有三位开局均汇聚在人界。

### 5.2 人界篇主角 —— 无名修士

- **身份**：凡人，有灵根被宗门选中
- **自定义范围**：全部（姓名、样貌、性别、性格、灵根、先天属性、修炼方向）
- **默认模板**：男性，16岁，中品灵根，剑修，偏远小镇出身
- **叙事定位**：完全空白状态进入世界，代入感是核心体验
- **差异化**：被欺骗者——对世界一无所知，真相随剧情揭开

### 5.3 灵界篇主角 —— 背叛的操偶师

- **身份**：紫微姬氏在灵界的暗线代理人，操控种族矛盾、管理情报网络。在灵界有精心构建的公开身份
- **自定义范围**：中等（姓名、样貌、性格、公开身份伪装、职业手段可调；"前仙界操偶师"核心身份固定）
- **默认模板**：女性，人族外表，法修（幻术与心灵类）辅修鬼道（情报审讯），表面温婉内心冷硬
- **叙事定位**：知情者——比任何人都清楚灵界真相，但这一次选择剪断线
- **差异化**：欺骗者——曾经操纵一切的人选择了反叛，但手段中的冷硬和残忍不会消失

**开局事件 —— 命运的讽刺**：

姬氏下令搜寻具有"天感"天赋的灵童——灵族极其罕见的天赋，能直接感知天道意志和灵气本源流动，可能看穿仙界操纵灵界的整个棋局。必须在天感觉醒前找到，带回仙界或抹杀。

主角搜寻数日未果。期间接到另一任务——潜入万劫域妖族领地刺探妖族鹰派与嬴氏的秘密交易。完成刺探撤离时被旧日仇家伏击，身受重伤，倒在妖族领地边缘废墟中。

醒来时发现躺在破旧窝棚里，伤口被草草包扎。照顾她的是一个妖族小女孩——蓬头垢面、衣衫褴褛的乞丐。女孩不会（或不愿）说话，但手脚勤快，每天送水、换药、找吃的。主角用幻术掩盖人族气息。

两人相处数日。这是主角数百年来第一次以"普通人"身份被对待。女孩在主角疼得睡不着时静静坐在旁边，把仅有的半块干粮递过来。

某天夜里，主角感觉到一股异常灵气波动——微弱却纯净，像天道本身在呼吸。循着波动看去，小女孩蜷缩在角落熟睡，周围灵气以不自觉的方式缓缓流动。

**那个照顾了她的妖族小乞丐，就是她找了数日未果的天感灵童。**

主角清楚孩子落入姬氏手中的下场：天感被观星者榨干利用，然后像所有用完的工具一样被处理掉。

她用仅剩的资源换取一块**破解石**（能撕开界壁从灵界直坠人界的禁术材料，代价是使用者灵力大幅损耗），带着女孩穿过破碎的界壁，坠入人界。

**人界困境多重叠加**：
- **照顾幼童**：不懂修炼不懂人界规矩的妖族女孩，随时可能暴露
- **隐藏妖族身份**：人界对妖族敌视，暴露则正道宗门追杀
- **躲避追踪**：姬氏暗探遍布灵界，破解石使用痕迹迟早被发现
- **灵力受损**：破解石代价加人界稀薄灵气，实力被大幅压制
- **仇家未除**：灵界伏击她的仇家不会因她消失就放弃

### 5.4 仙界篇主角 —— 青帝

- **身份**：散修传奇，从人界修炼至大罗金仙的隐世大能
- **自定义范围**：最低（仅可修改姓名）
- **固定设定**：男性，符道主修/阵道辅修，木火双灵根，称号"青帝"
- **性格**：聪慧机敏洒脱不羁，表面轻浮不拘小节，经历八次金仙劫后精疲力竭
- **绝学**：十二天符——将大罗金仙级灵力封入符箓，每张足以重创同级别大能
- **叙事定位**：传奇落幕——站在三界顶点，故事关于如何面对注定的终结
- **差异化**：明知者——知道天道真相、修炼代价、金仙劫无解，但选择最疯狂的路：散尽一切重新来过

**开局事件 —— 第九次金仙劫**：

青帝独站在苍冥蛮荒深处，天空变色——天道本身在注视他。第九次金仙劫降临。

**第一条路：硬抗金仙劫**——雷劫强度远超前八次之和，正常不可能战胜，死亡是默认结局。

**第二条路：置之死地而后生**——在渡劫同时将全部灵力封入十二张符箓。需在雷劫间隙完成封印，容错率几乎为零。成功则十二符箓完成（每张蕴含重创大罗金仙之力），但自身修为尽散跌落凡人。天劫因目标"已死"消散。青帝以凡人之躯回到人界从头修炼，手握十二张足以改变三界格局的底牌。

**散功后记忆缺失（重要机制）：**

封存灵力的过程同时封存了附着在灵力上的大量记忆——数万年的修炼感悟、经历、情感随灵力一起被剥离。散功后青帝以**记忆残缺**的凡人之躯存于人界。

| 保留 | 丢失 |
|------|------|
| 自己叫李云霄，道号"青帝" | 五色帝的其他四人（完全不知道他们的存在） |
| 自己曾是大罗金仙 | 出身、成长经历、修炼历程 |
| 在第九次金仙劫中将灵力封入十二焚炎符 | 焚炎符的真正威力（只知有12张符但不知有多强） |
| 基础修炼理论（像读过教科书但没有实感） | 天道收债的真相、修炼的本质、天劫的机制 |
| 自己选择了"散尽一切重新来过" | 为什么做这个选择、具体规划是什么 |

**记忆随修为恢复：**

| 修为阶段 | 恢复的记忆 |
|---------|-----------|
| 练气期 | 隐约觉得自己"很擅长画符"，感知到识海中有12张符箓存在但无法触碰 |
| 筑基期 | 能清晰感知焚炎符的存在，尝试以神识激发却力不从心（神识不足以支付80点消耗），只浮现封符时雷光中的碎片画面 |
| 金丹期 | 神识达标首次使用焚炎符，震惊于大罗级力量，恢复"天劫是天道收债"的核心理解 |
| 元婴期 | 恢复五色帝赌局的完整记忆——白帝的沉默、赤帝的大笑、黄帝的点头、黑帝的低语 |
| 化神期 | 恢复所有记忆——同伴的死、黄帝与黑帝的失踪、太昊的九劫、自己选择散功的全部理由 |

**设计意图：** 青帝和玩家知道的一样多，代入感最大化。恢复记忆的过程就是探索三界真相的过程。化神期（人界修为上限）恰好恢复全部记忆，此后飞升灵界面对太昊、寻找黄帝和黑帝。焚炎符的神识门槛（80点）构成叙事与机制的统一——筑基期感知但不解渴，金丹期首次释放成为记忆恢复的关键转折点。

|------|------------|------|

## 六、五色帝 —— 五行至极的五个赌徒

### 6.1 总论

五色帝是末法劫后期至寂灭劫初期，三界最强的五位大罗金仙。他们分别以五行至极为道，并非统治集团（那是太昊），而是各自为政的顶级修士。

**五色帝不是称号，是一场赌局的五个赌徒。**

末法劫末期至寂灭劫初期（~112,000年），五位大罗金仙在苍冥醉仙楼偶遇。因各自对金仙劫的不同理解，赌了一个约——"谁能找到渡过金仙劫的方法"。五人各持己见、分道扬镳，约定每万年聚一次分享发现。

这场赌局持续了约八万年。两个人赌输了——死在了天劫里。一个人赌赢了另一种答案——散尽修为重新来过。两个人还在赌，只是赌注从"如何活"变成了"如何让三界活"。

### 6.2 苍冥赌局

赌局发生于绝对年份约112,000年（寂灭劫初期），苍冥醉仙楼。

- **赤帝（拍桌）**："天劫的雷电是火！我能炼！"
- **白帝（沉默）**："……试试。"
- **青帝（灌酒）**："你们都没我疯。散了重来。"
- **黄帝（点头）**："灵气会回来的。我帮它。"
- **黑帝（低声）**："我先看看规则。"

五人各以自己的道应对金仙劫：

| 帝 | 五行 | 方向 | 赌注 | 结果 |
|----|------|------|------|------|
| 白帝·凌霄 | 金/西 | 剑修/器修 | "极致的锋芒可以斩破天劫" | 第六劫陨落 |
| 赤帝·炎无极 | 火/南 | 丹修/法修 | "天劫之力可以炼化为灵气回馈天地" | 第七劫陨落 |
| 青帝·李云霄 | 木火/东 | 符修/阵修 | "散尽一切重新来过" | 散功归凡，记忆缺失 |
| 黄帝·坤元 | 土/中 | 阵修/体修 | "天劫收的是灵气，让灵气回来" | 存活，隐匿人界 |
| 黑帝·渊寂 | 水/北 | 法修/鬼修 | "看清天劫的一切规则，找到漏洞" | 存活，潜伏灵界 |

### 6.3 各帝详设

#### 白帝·凌霄 —— 金/西 【已陨落】

> *"天劫是斩不断的——这话我同意。但我偏要试试，能斩几刀。"*

| 项目 | 设定 |
|------|------|
| **灵根** | 金系天灵根 |
| **道途** | 剑修主修 / 器修辅修 |
| **出身** | 灵界万劫域妖族散修（金灵根在妖族极罕见，从小被排挤） |
| **性格** | 沉默如剑，行动如刃。对战斗有近乎病态的执着，但不是嗜血——是把战斗当作"对话" |
| **陨落** | ~142,000年，第六次金仙劫。拔剑迎天劫，本命剑"凌霄"崩碎，以碎剑残片继续战斗七日七夜，灵力耗尽而亡。**黑帝在场未援手**——判断出手代价过大，此事成为其毕生愧疚 |
| **遗产** | 剑意碎片散落三界，最大碎片落在人界东荒断剑之地。天道剑宗在此建宗，不知脚下踩的是白帝遗泽。铸造术残篇《庚金铸剑录》散落，嬴氏兵器铸造传统疑似源自其旁支传承 |

#### 赤帝·炎无极 —— 火/南 【已陨落】

> *"天劫的雷电不过另一种火。既然是火，我就能炼。"*

| 项目 | 设定 |
|------|------|
| **灵根** | 火系天灵根 |
| **道途** | 丹修主修 / 法修辅修 |
| **出身** | 人界飞升散修丹师。在人界灵气稀薄时靠炸炉灵气反噬强行吸纳修炼，九死一生 |
| **性格** | 狂傲而真诚，说话声音洪亮，笑起来能掀房顶。五帝中脾气最暴、人缘最好——从不记仇 |
| **陨落** | ~157,000年，第七次金仙劫。以太古丹炉炼化天劫，前六道雷劫成功炼化灵气回馈天地，第七道强度超限丹炉炸裂。以肉身为炉、以命火炼劫，**丹成人亡**。最后炼化的灵气回馈大地，人界灵气从0.35微升至0.40——灵气复苏从此刻开始 |
| **遗产** | 太古丹炉炸裂为三片，丹霞得最大残片。赤帝丹经三界丹修至高典籍，完整版失传，丹霞只存六成。其中最关键的"炼劫化灵"三篇与赤帝一同灰飞烟灭。临终前将遗言玉简托黑帝转交青帝，黑帝已带在身上超过四万年 |

#### 黄帝·坤元 —— 土/中 【存活·隐匿人界】

> *"灵气在回来。很慢，但它确实在回来。我替它守着就是了。"*

| 项目 | 设定 |
|------|------|
| **灵根** | 土系天灵根 |
| **道途** | 阵修主修 / 体修辅修 |
| **出身** | 人界散修。在灵气枯竭处站了三百年，直到感觉大地深处灵气脉动，开创"地脉阵法" |
| **性格** | 闷到骨子里。别人说话他点头，别人争论他沉默，别人打起来他挡在中间。说到做到 |
| **当前状态** | 赤帝陨落后（~157,000年），黄帝自封修为至化神期，沉入人界中州灵脉最深处。以"老坤"之名在灵脉坊市周边活动，买灵草种子种在灵脉枯竭处——是真的在种地。人界灵气从0.35回升至0.40，表面是赤帝陨落灵气回馈，实际上黄帝暗中修复功不可没 |
| **与青帝** | 不知道青帝已散功。上一次见面是赤帝陨落后，青帝说要"回混沌虚空准备第九次金仙劫"。黄帝不知道身边是否还会再见到那个吊儿郎当的酒鬼 |

#### 黑帝·渊寂 —— 水/北 【存活·潜伏灵界】

> *"我看了一切。不是因为我好奇，是因为我欠他们的。"*

| 项目 | 设定 |
|------|------|
| **灵根** | 水系天灵根 |
| **道途** | 法修主修 / 鬼修辅修 |
| **出身** | 灵界鬼族与人族混血。半人半鬼，天生同时看到生者与死者的世界，对灵气本质有独到感悟 |
| **性格** | 深不可测，但不是刻意神秘——是真的不知道怎么表达。他看到的东西太沉重，语言变得极其精简 |
| **当前状态** | 赤帝陨落后（~157,000年），离开仙界潜入灵界冥渊。以鬼族"渊老"身份混入冥渊长老会。以天渊镜碎片记录灵界种族矛盾、仙界操纵手法、灵气复苏规律，将信息封入碎片藏于灵界各处 |
| **与青帝** | **他知道青帝还活着。** 黑帝是天渊镜主人，水行灵力可感知三界灵气大规模流动。青帝散功时的大罗级灵力海啸不可能逃过他的感知。他带着赤帝遗言玉简等了超过四万年，现在在等青帝恢复到足以接收这些信息的时候。他选择了等——因为他不确定一个失忆的凡人能否承受玉简中的信息 |

### 6.6 三位主角与五色帝的交汇
