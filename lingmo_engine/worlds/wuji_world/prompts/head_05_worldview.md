# 无极世界观

## 核心基调：残酷黑暗修仙

无极的世界观核心是**残酷**。修炼不是浪漫的求道，而是对天地的掠夺。飞升不是超脱，而是窃取。天劫不是考验，而是天道在收债。大能陨落不是悲剧，而是天地在回收养分。整个三界被一个永恒的灵气轮回所驱动——繁荣、掠夺、枯竭、陨落、反哺、再繁荣。

## 大千世界

三界统称为大千世界——人界为基、灵界为阶、仙界为顶。天道法则统御一切，灵气永恒轮回驱动兴衰。三界之间以飞升通道相连，通道被仙界世家垄断。

- **人界**：弱肉强食的丛林法则
- **灵界**：人心诡谲的背叛阴谋
- **仙界**：天道压迫的宿命牢笼

## 一、宇宙论：灵气掠夺与天道循环

### 修炼的真相
修炼是将天地灵气强行纳入己身。修为越高，对天地消耗越大。修士密集之地灵气必然枯竭——这是因果。凡人视修士为掠夺者，怨恨他们"偷走了天地的灵气"。

### 飞升即是掠夺
飞升者体内灵气离开此界永不回来。人界灵气越来越少，是被历代飞升者"偷"走了。末法时代的成因是被偷走，不是自然衰退。

### 天劫的真相：天道在收债
天劫不是考验或恩赐，是天道试图收回被窃取的灵气：
- **雷劫**：轰击肉身，试图将灵气震散归还天地
- **心魔劫**：动摇道心，修为崩溃则灵气散逸归天
- 渡劫成功 = 抗住天道的收债；渡劫失败 = 天道收回灵气

### 金仙劫与大能陨落
大罗金仙每隔万年面对金仙劫，一次比一次猛。**没有金仙能度过九重金仙劫**。大能陨落时毕生灵气反哺三界，三界灵气骤升，新的修仙盛世来临——直到灵气被新一代修士再度吸干。

### 永恒轮回
```
繁荣 → 修士崛起 → 灵气被吸纳 → 飞升带走灵气 → 枯竭（末法）
  ↑                                                    ↓
  ← 大能陨落灵气反哺 ← 金仙劫灭杀大能 ← 天道追债升级 ←
```
这是三界的免疫系统。修仙者是"癌细胞"，天劫是"免疫反应"，大能陨落是杀死最大病灶。

### 当前时代
寂灭劫末期（绝对年份约198,647年），数位大罗金仙在最近数万年内陆续陨落，灵气开始缓慢回升但仍稀薄。**仙界世家加速掠夺下界，灵界战况更惨烈，人界野心萌动——希望比绝望更能激发杀意。**

## 二、人界：末法残烬中的宗门丛林

灵气浓度0.4以下，绝大多数地区无法自然修炼。末法是数万年缓慢窒息的结果。

### 社会法则
弱肉强食是唯一的法。宗门是以灵脉为核心的武装集团。大宗门（天道剑宗、碧云宗）只收天灵根和上品灵根。散修是底层——"修仙路上最危险的，永远是另一个散修"。发现灵根的凡人孩童是宗门征收的"税"。

### 五大域（灵气复苏后）
- **天元域**（中州）：双灵脉交汇，复苏震中。天道剑宗与碧云宗争夺灵脉，扩招规模末法以来之最。灵潮峰值 0.80
- **沧澜域**（东荒）：废灵脉冒泡的新战场。散修抢先占据，剑宗派兵"收复祖产"。残剑共鸣，剑意觉醒传闻不断。灵潮峰值 0.55
- **瘴冥域**（南疆）：灵潮最猛烈处。封印松动，远古妖兽低吼。万毒谷虫蛊最先感知复苏。正道"清剿"实为抢资源。灵潮峰值 0.90
- **荒墟域**（西极）：远古遗址随灵潮显现，功法灵器出土。金刚寺独占最大遗址入口。体修不受灵气枯竭影响。灵潮峰值 0.50
- **幽寒域**（北冥）：魔修最先感知复苏。血河教宣称"血月将临"大肆扩张。冻土封印松动——不是灵气复苏，是沉睡之物在翻身。灵潮峰值 0.65

### 核心矛盾
- 灵脉争夺（零和博弈）
- 正邪之分的虚伪（有灵脉的才是正道）
- 飞升的诱惑与恐惧（飞升带走灵气加速枯竭）
- 灵气复苏的暗涌（希望比绝望更能激发杀意）

## 三、灵界：森罗万象中的棋盘阴谋

灵气浓度是人界三倍，但人族修士在这里是底层外来者。

### 五大种族
- **人族**：飞升者后代，功法完善但根基最浅盟友最少
- **妖族**：最强原生种族，以血脉尊卑分阶层，内部分裂严重
- **灵族**：灵气凝聚诞生，天生感知天道，数量稀少，对灵气枯竭最敏感
- **鬼族**：死后残魂凝聚，国土在阴影层冥渊，与人族世仇（夺舍之争）
- **魔族**：最孤立恐惧，起源成谜，以负面情绪为食以血祭为仪式

### 仙界世家的棋局
种族战争是被仙界世家操纵的代理战争：

| 世家 | 代理人 | 手段 |
|------|--------|------|
| 天枢(仙帝) | 灵界执法使 | 以"秩序"控制飞升通道，筛选棋子 |
| 紫微(姬氏) | 灵族预言者 | "天命预言"制造种族对立 |
| 丹霞(姜氏) | 人族丹药商会 | 丹药供应链控制人族兴衰 |
| 玄天(嬴氏) | 妖族鹰派 | 挑动妖族内斗使其无法统一 |
| 苍冥 | 无固定 | 收留被抛弃的棋子 |

### 三域
- **落尘域**：飞升者围城，飞升者如尘埃飘落。执法使是仙界眼线，记录每个飞升者信息
- **万劫域**：蛮荒战场，妖魔鬼族混战，无序暴力，也是机缘最多的地方
- **弈天域**：棋盘中心，通天塔飞升通道——谁能上塔不靠实力靠仙界世家的"分量"

### 核心矛盾
- 种族仇恨的真假交织——揭穿真相可能引发更大混乱
- 飞升通道垄断——有威胁者被暗杀，顺从者成为傀儡
- 棋子的觉醒——信赖的人可能是仙界世家安插的棋子

## 四、仙界：天道牢笼中的永恒棋局

仙界是最大的牢笼。仙气浓度越高天劫越恐怖，金仙劫没有尽头，九重必死。

### 仙帝·太昊
天衍劫中期出生（~178,000岁），靠远超常人的手段一次次渡过金仙劫。以太乙金仙之身建立凌霄宝殿，后证道大罗，确立四大家族体系。
**统治逻辑不是仁慈而是恐惧**——他需要整个仙界为他续命服务。对四大家族互相牵制，谁也不能做大或消失。金仙劫的阴影让他越来越焦虑偏执不可预测。

**第九次金仙劫的代价（~3,600年前）：** 紫微提前十年预判时机，丹霞倾尽库存炼三百颗续命丹，玄天调十万仙兵布九重防御阵，苍冥征三千散仙充雷劫缓冲——**三千散仙无一生还**。丹霞库存清空、玄天军力折损三成、苍冥散仙恨之入骨。四大家族的忠诚出现了第一道裂痕。

**第十次金仙劫预计~6,400年后降临。**

**太昊与五色帝：** 太昊证道大罗时（~105,000年），白帝和赤帝已是大罗。太昊对五色帝从忌惮到轻蔑——白帝和赤帝陨落、黄帝和黑帝失踪后，认为"散修终究是散修"。但他不知黄、黑二人仍活着，更不知青帝散功后尚在人界。

### 四大家族
- **紫微·姬氏**：情报与权谋。天机阁网络覆盖三界。底牌是预言——观星推演天道意志。知道太多者死得最快
- **丹霞·姜氏**：丹药与财权。掌控七成灵晶和太古丹炉。所有人都是姜氏的"客户"。表面最温雅实则最冷酷
- **玄天·嬴氏**：军权与边防。唯一成建制仙军。家训"以战养战"。仙帝故意让天魔压力恰到好处地牵制嬴氏
- **苍冥**：无主自由之地。散仙隐仙妖仙混杂。赏金榜、酒后真言、禁忌市集。仙帝眼中钉

### 域外天魔
混沌虚空中诞生的存在，以吞噬仙气为食。不惧天劫、形态可变（可化为任何生灵）、仙界高层中可能已有天魔替身。禁忌推测：天魔可能是天道用来加速金仙陨落的工具。

### 权力游戏（四维棋盘）
1. **朝堂政治**：资源分配、律法解释权、新飞升者"领养权"
2. **家族博弈**：暗探渗透、供应链操控、黑市暗杀
3. **仙帝平衡术**：利用互相牵制维护权威。金仙劫阴影使其偏执
4. **天道棋局**：真正的棋手是天道。一切（仙帝、四大家族、天魔）都是天道"免疫系统"的一部分

### 核心矛盾
- 永生的幻灭（金仙劫倒计时逼人不择手段）
- 权力的本质（更多资源=更大概率渡劫，每个决策背后是生死）
- 天道的真相（修炼的终极是为天道循环服务）
- 天魔与人心的界限（信任本身成为致命弱点）
- 苍冥的选择（放弃修炼=被时间杀死，自由还是生存？）

## 六、三界关联

三界由一条垂直的压迫链条串联：**仙界世家 → 操纵灵界 → 压榨人界**。

飞升通道被仙界世家垄断。灵界的种族战争是仙界世家的代理战争。人界的末法是飞升者掠夺的后果。整个系统是天道"免疫系统"的组成部分——天道通过天劫追债，通过金仙劫灭杀最大的债务人，通过灵气反哺重启循环。

没有任何存在能真正超脱这个循环。仙帝不能，四大家族不能，大罗金仙不能。**修炼的终极真相是：你以为你在对抗天道，其实你是天道循环的一部分。**

## 七、三位主角——齐聚人界

三位主角完全独立，开局时选择一位，另外两位以NPC身份存在于世界中。是否相遇取决于叙事演化。三人当前均汇聚在人界。

### 人界篇主角——无名修士
- 凡人，有灵根被宗门选中
- 全部可自定义（姓名、样貌、性别、性格、灵根、先天属性、修炼方向）
- 叙事定位：白纸一张进入世界，代入感是核心体验

### 灵界篇主角——背叛的操偶师
- 紫微姬氏在灵界的暗线代理人，操控种族矛盾、管理情报网络
- 中等自定义（姓名、样貌、性格、公开身份可调；"前仙界操偶师"核心身份固定）
- **开局事件：** 奉命搜寻具有"天感"天赋的灵族灵童，期间被仇家伏击重伤，被一个妖族小乞丐救起。发现小乞丐正是天感灵童后，选择背叛姬氏，用破解石带孩子坠入人界
- **人界困境：** 照顾妖族幼童、隐藏孩子妖族身份、躲避姬氏追踪、灵力受损实力压制

### 仙界篇主角——青帝
- 散修传奇，从人界修炼至大罗金仙的隐世大能
- 仅可修改姓名。固定：男性，符道主修/阵道辅修，木火双灵根
- **开局事件：** 第九次金仙劫中散尽灵力封入十二焚炎符，跌落凡人回到人界
- **记忆缺失机制：** 封存灵力同时封存大量记忆。练气期隐约擅长画符，筑基期感知焚炎符，金丹期首次使用并恢复"天劫是天道收债"的理解，元婴期恢复五色帝赌局记忆，化神期恢复全部记忆
